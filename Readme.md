# view-image-size

A in-browser module to get dimensions (i.e. width, height, type) of an image file (by using a DataView)

## Supported formats

* BMP
* CUR
* DDS
* GIF
* ICNS
* ICO
* JPEG
* KTX
* PNG
* PNM (PAM, PBM, PFM, PGM, PPM)
* PSD
* SVG
* TIFF
* WebP

## Installation

```shell
npm install view-image-size --save
```
